# 电视剧营销投流 AI 工作台 - 一键部署指南

## 方案一：Render 一键部署（推荐 ⭐）

### 步骤：

**第1步：登录 Render**
1. 打开浏览器，访问：https://dashboard.render.com/login
2. 点击 **GitHub** 按钮登录（用你的 GitHub 账号 `wubinghuang2019-source`）

**第2步：一键部署后端**
1. 登录成功后，直接在浏览器地址栏粘贴这个 URL：
```
https://render.com/deploy?url=https://github.com/wubinghuang2019-source/drama-adflow-ai
```
2. Render 会自动读取你的仓库和 `render.yaml` 配置
3. 只需要填一个环境变量：
   - Key: `DASHSCOPE_API_KEY`
   - Value: 你的阿里云百炼 API Key（暂时没有可以先不填）
4. 点击 **Apply** 按钮

**第3步：等待部署**
- 大约 3-5 分钟，Render 会自动构建和部署
- 部署成功后你会看到类似地址：`https://drama-adflow-api-xxxx.onrender.com`
- **把这个地址复制下来发给我**

## 方案二：如果方案一不行，手动部署

### 第1步：登录 Render
1. 访问：https://dashboard.render.com
2. 用 GitHub 登录

### 第2步：新建 Web Service
1. 点击 **New** → **Web Service**
2. 如果看不到仓库，点击 **"Configure account access"** 或 **"Connect more repositories"**
3. 搜索 `drama-adflow-ai`，点击 **Connect**

### 第3步：配置（重要！）
- **Name**: `drama-adflow-api`
- **Environment**: `Python 3`
- **Build Command**: `pip install -r api/requirements.txt`
- **Start Command**: `cd api && gunicorn server:app --worker-class sync --workers 1 --bind 0.0.0.0:$PORT`
- **Instance Type**: `Free`

### 第4步：环境变量
点击 **Advanced** → **Add Environment Variable**：
- Key: `DASHSCOPE_API_KEY`
- Value: 你的阿里云百炼 API Key

### 第5步：点击 Create Web Service
