## 🔧 第二步：部署后端到 Render

### 1. 注册 Render 账号
- 访问：https://dashboard.render.com/
- 使用 GitHub 账号登录

### 2. 创建 Web Service
1. 点击 "New +" → "Web Service"
2. 选择你的 GitHub 仓库 `drama-adflow-ai`
3. 配置：
   - **Name**: `drama-adflow-api`（或自定义）
   - **Region**: `Singapore`（亚太地区）
   - **Branch**: `main`
   - **Root Directory**: 留空
   - **Runtime**: `Python 3`
   - **Build Command**: `pip install -r api/requirements.txt`
   - **Start Command**: `gunicorn api.server:app --worker-class sync --workers 1 --bind 0.0.0.0:$PORT`
   - **Instance Type**: `Free`

### 3. 设置环境变量
点击 "Advanced" → "Add Environment Variable"：
- **Key**: `DASHSCOPE_API_KEY`
- **Value**: 你的阿里云百炼 API Key

### 4. 部署
1. 点击 "Create Web Service"
2. 等待 5-10 分钟部署完成
3. 部署成功后，你会得到一个 URL：
   - 例如：`https://drama-adflow-api.onrender.com`

### 5. 测试后端
访问：`https://你的域名.onrender.com/health`
- 如果返回 `{"status": "ok", "message": "电视剧营销投流 AI 工作台运行正常"}`，说明部署成功

## 🎨 第三步：部署前端到 Vercel

### 1. 注册 Vercel 账号
- 访问：https://vercel.com/
- 使用 GitHub 账号登录

### 2. 导入项目
1. 点击 "Add New..." → "Project"
2. 选择你的 GitHub 仓库 `drama-adflow-ai`
3. 配置：
   - **Framework Preset**: `Other`
   - **Root Directory**: `./`（默认）
   - **Build Command**: 留空
   - **Output Directory**: `./`（默认）

### 3. 部署
1. 点击 "Deploy"
2. 等待 1-2 分钟部署完成
3. 部署成功后，你会得到一个 URL：
   - 例如：`https://drama-adflow-ai.vercel.app`

## 🔍 常见问题排查

### 问题 1：后端部署失败
**可能原因**：
- API Key 未设置
- 依赖安装失败

**解决方法**：
1. 检查 Render 的环境变量设置
2. 查看 Build Logs 和 Deploy Logs
3. 确认 `requirements.txt` 格式正确

### 问题 2：前端无法连接后端
**可能原因**：
- API 地址未修改
- CORS 配置问题

**解决方法**：
1. 检查 index.html 中的 API 地址
2. 确认后端已添加 `flask-cors` 支持

### 问题 3：生成内容失败
**可能原因**：
- API Key 无效
- 阿里云百炼服务未开通

**解决方法**：
1. 检查 API Key 是否正确
2. 确认阿里云百炼服务已开通
3. 检查 Render 日志中的错误信息
