# 电视剧营销投流 AI 工作台

一站式电视剧营销投流工作台，支持策略生成、文案生成、数据监控、效果分析四大核心功能。

## 🎯 功能模块

### 1. 策略生成
- 输入剧集基本信息，自动生成完整投流策略
- 包含目标受众画像、平台内容策略、预算分配、投放节奏规划

### 2. 文案生成
- 一键生成抖音、小红书、视频号、微博多平台文案
- 支持多种文案风格选择

### 3. 数据监控（开发中）
- 实时监控各平台投放数据
- 自动生成日报/周报

### 4. 效果分析（开发中）
- 素材效果分析
- 用户行为分析
- ROI 计算
- 优化建议生成

## 🛠️ 技术栈

- **前端**: HTML + JavaScript（单页应用）
- **后端**: Python Flask + 流式响应
- **AI 模型**: 阿里云百炼 qwen-max
- **部署**: Render（后端）+ Vercel（前端）

## 📦 部署指南

### 后端部署（Render）

1. Fork 本仓库
2. 在 Render 创建新的 Web Service
3. 连接 GitHub 仓库
4. 设置环境变量：
   - `DASHSCOPE_API_KEY`: 阿里云百炼 API Key
5. 部署完成

### 前端部署（Vercel）

1. 在 Vercel 导入 GitHub 仓库
2. 设置 Root Directory 为 `/`
3. 部署完成

## 🎨 使用说明

1. **策略生成**: 填写剧集信息 → 选择投放平台 → 点击生成
2. **文案生成**: 输入剧集片段 → 选择文案风格 → 生成多平台文案

## 📝 注意事项

- 需要配置阿里云百炼 API Key
- 首次生成可能需要等待 10-20 秒
- 建议使用 Chrome 浏览器访问
